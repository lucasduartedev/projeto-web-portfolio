Sobre o HTML

O HTML é uma sigla para hypertext markup language, que significa em portugês linguagem de marcação de hipertexto, é uma linguagem de marcação padrão utilizada para estruturar as páginas da WEB.

O HTML utiliza "tags" para organizar elementos como textos, links, formulários e imagens, aplicando semântica e organização na estrutura das páginas redenzadas pelos navegadores como Google Chrome e Firefox.


CSS

CSS - Cascading Style Sheets ou Folhas de Estilo em Cascata é uma linguegem de estilo é usada para dar cores, definir layout e a formatação das páginas.


Javascript

Ou simplismente JS, é a única linguagem de programação interpretada pelos navegadores com o objetivo de aplicar interatividade, trazer dinamismo e experiências mais ricas para os usuários.


Desenvolvimento WEB

Uma simples página pode ser desenvolvida utilizando apenas HTML, mas ficaria simples demais. Por isso ao implementar HTML, CSS e Javascript em conjunto, temos uma página mais rica em detalhes e interações.

As páginas WEB não são desenvolvidas apenas com essas três tecnologias, a exemplo de sites que é necessário realização de autenticação através de login, temos outras linguagens de programação como Java, PHP e etc e banco de dados relacionais (SQL) e não relacionais (NoSQL) para armazenar as informações de acesso e seus conteúdos.
